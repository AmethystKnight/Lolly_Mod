Info
==========
This is a mod for Minecraft 1.21.1, requiring NeoForge. All graphics are temporary images taken from the internet.

To play the mod click [lollymod-1.0.0.jar](https://raw.github.com/AmethystKnight/Lolly_Mod/blob/master/lollymod-1.0.0.jar) and download the file, then put it in your Minecraft mod folder.
Alternatively you can donwload/clone the repo and compile the mod yourself with `gradlew build` after which you can find the jar in build/libs.

Features
==========
* A lolly item created by smelting sugar in a furnace or smoker
* A lolly block crafted using 9 lollies in a crafting table
* A Candy Cane crafted with 1 lolly and 2 sticks in a crafting table using the same layout as a spear, eating the  Candy Cane gives jump boost and speed
* All items can be accessed through the lolly tab in the creative menu
